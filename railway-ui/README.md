# PakRail Frontend (React)

This is the user interface for the Online Railway Ticket Reservation System.

## 🚀 Development Setup

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Dev Server**:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:3000`.

## 🎨 Technology Stack
- **React**: Core library
- **Vite**: Build tool & Dev server
- **React Router**: Client-side routing
- **Lucide React**: Premium iconography
- **Vanilla CSS**: Global styling and glassmorphism utilities
- **Axios**: API communication

## 📂 UI Features
- Responsive Navigation
- Train Search with Date Selection
- Modern Login/Register forms
- PNR Status dashboard
- Admin Train Management view
